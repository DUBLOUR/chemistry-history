Инструкция по установке оффлайн-версии

  1. Укажите папку, в которой будет храниться интерактивная таблица.
    Если вам не предлагалось выбрать путь то файл уже скачан; нажмите Ctrl+J чтобы открыть папку загрузки.
  2. Нажмите правой кнопкой мыши по файлу и нажите «Распаковать архив в текущую папку».
  3. В создавшейся папке откройте любым современным браузером файл index.html
    Если вашим браузером по-умочанию является Internet Explorer, то нажмите на файл правой кнопкой мыши и найдите пункт «Открыть с помощью...», где выберите подходящий вариант.
  4. Теперь всегда обращайтесь к index.html когда нужно будет запустить таблицу в оффлайн-режиме.
